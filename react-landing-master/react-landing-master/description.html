React landing, a SaaS/Product landing page with minimal design and feel. React Landing is built with React Material UI & create react app. Each page section is composed of its own jsx and SCSS files, organized in components & sass directory and So it’s easy to customize and reuse.

IMPORTANT: It’s not a regular HTML landing page. It’s built with React.

React Landing Features

Customizability

React Landing gives you full control of customization. A developer friendly landing template which gives you clean code and excellent code structure.

Regular update

Get unlimited future updates on purchase. React Landing will be up to date with future react versions.

Git Repository

Access git repository and keep an eye on update process and changes. Request Github access

Full responsive

React landing is compatible with any device. Material UI Grid makes everything is much easier. To define Grid size, spacing, and breakpoints you just need to set properties to Grid components.

Material UI

High quality React material UI components are used. Buttons, Cards, Icons, Inputs are heavily used in this template.

Multiple Versions

Alternative versions of sections included for different purpose of usages. You can create your own version by rearranging components.

Sass support

Styles are written in scss files for easy customization. You can add custom css and import third party style files and extend the design easily.

Scroll to navigation

One page scrolling & Link navigation. Reusable compoent <SrollTo to="homeId">Home<SrollTo> allow you to use scroll navigation feature anywhere in the template.

Carousels

Carousels are made with swiper, a responsive react carousel package.
